Method

Each video model in the catalog is run on a fixed prompt set per dataset (humaneval / wmb / video_phy_2 / physics_iq). Outputs are scored by multiple evaluators (Claude, Gemini, GPT, Qwen-9B/27B/397B, our finetuned checkpoints) under several scoring schemas (plain/v1, plain/v2, subq_hint/v1, …) producing a per-prompt gen_avg (general fidelity) and phys_avg (physical correctness) score, plus per-physical-law scores. The leaderboard shows the newest coverage > 0 result per (model, dataset, subset, evaluator, schema); older runs are kept in each row's history.

Aggregate gen_avg/phys_avg are domain-pooled: arithmetic mean over per-prompt scores. Macro-averaging across physical laws produces different rankings; we ship the domain-pooled view because it matches the upstream registry.
